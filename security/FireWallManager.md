## TLDR

Centraly manage firewall config and rules for [[AWSOrganizations]]

## Supported Services

- [[WAF]].
- [[Shield]] Advanced.
- [[EC2]] Security Groups.
- [[NetworkFirewall]]. (VPC level)
- [[Route53]] DNS Firewall.

## Features

- Region level.
- **Automatic apply rules to new resources.**
- Security Policy -> common set of security rules.
